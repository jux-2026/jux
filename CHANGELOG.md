# Changelog

All notable changes to Jux will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [Unreleased]

## [0.1.2] - 2026-07-18

### Fixed

- Windows release branding supports cargo-dist archives with a flat layout.

## [0.1.1] - 2026-07-18

### Added

- npm installation through `@jux-2026/jux` with channel-specific binaries and
  GitHub OIDC trusted publishing.

## [0.1.0] - 2026-07-13

### Added

- Initial local agent runtime, CLI, and TUI foundation.
- GitHub Release packaging with shell and PowerShell installers.
